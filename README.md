# green_dp

디스플레이 전용(로그인/입력 없음) 대시보드 사이트입니다.

## 특징
- **로그인 완전 제거**
- **`/school`(입력/편집) 제거**
- 상단 **드롭다운으로 학교 선택** 후 대시보드 데이터를 조회

## 실행

```bash
npm install
npm run dev
```

## 주의
- 학교 목록 로드/데이터 조회는 Firestore 읽기 권한(규칙)에 따라 동작합니다.
- 운영/공개 시에는 노출 범위에 맞게 Firestore Rules 또는 서버 API 방식(권장)으로 접근을 통제하세요.
